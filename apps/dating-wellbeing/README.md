# Dating Wellbeing Analyzer

**A SAFE Framework App**

Privacy-first dating profile analysis with red flag detection and pattern learning.

## What It Does

- Profile analysis from screenshots or text
- Red flag detection (love bombing, control, age gaps)
- Pattern learning with consent
- Reflection prompts for better decisions
- 96% local processing

## SAFE Integration

Session-based consent model with two data streams:
1. **Profiles** (session retention) - deleted after use
2. **Patterns** (permanent retention) - optional learning

pip install -e .
